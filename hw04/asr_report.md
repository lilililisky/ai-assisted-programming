# 开源语音识别（ASR）方案对比报告
## 一、调研方案
本次调研三种主流开源ASR方案：OpenAI Whisper、Vosk、FunASR。

## 二、对比维度
| 对比维度         | OpenAI Whisper                          | Vosk                                  | FunASR                                  |
|------------------|-----------------------------------------|---------------------------------------|-----------------------------------------|
| 许可协议         | MIT License（商业友好）                 | Apache-2.0 License                    | MIT License                             |
| 语言与方言支持   | 99种语言+方言，支持中英混合             | 多语言，支持小语种/方言                | 支持中英日等，方言支持较丰富             |
| 模型体量         | 多尺寸：tiny/base/small/medium/large    | 轻量模型（<100MB）到大型模型          | 支持轻量/大型模型，可按需选择           |
| 推理速度         | 中等，base模型在PC上流畅运行            | 快，轻量模型适合边缘设备              | 快，优化了实时推理速度                  |
| 流式/实时支持    | 需第三方扩展（如faster-whisper）        | 原生支持流式/实时识别                  | 原生支持流式/实时识别                    |
| 依赖与部署难度   | 低，Python一键安装，依赖ffmpeg           | 中，需下载模型文件，配置路径          | 中，依赖PyTorch，需安装额外依赖          |
| 笔记本/PC实测感受 | 易用性高，文档完善，识别准确率高        | 轻量模型启动快，适合低资源设备        | 功能丰富，支持热词/标点，准确率优秀      |

## 三、选型理由
选择 **OpenAI Whisper** 作为本次实现方案，理由：
1. 易用性极高：Python `pip` 一键安装，无需复杂配置，适合快速验证；
2. 多语言支持完善：完美支持中文+英文混合场景，符合本次作业文稿内容；
3. 准确率稳定：`base` 模型在普通PC上即可运行，识别准确率满足作业需求；
4. 生态成熟：有大量第三方扩展（如`faster-whisper`），后续可优化速度。
